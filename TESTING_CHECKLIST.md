# Чек-лист тестирования: Динамические плейсхолдеры

## Предварительные требования

- [ ] Сервер запущен (`python manage.py runserver 3000`)
- [ ] Установлены библиотеки: python-docx, openpyxl, python-pptx
- [ ] Есть доступ к админ-панели или правам создания шаблонов

---

## 1. Тестирование интерфейса управления плейсхолдерами

### 1.1 Открытие формы создания шаблона
- [ ] Перейти в "Шаблоны документов" → "Создать шаблон"
- [ ] Найти раздел "Управление плейсхолдерами"
- [ ] Увидеть alert с автоматическими плейсхолдерами (дата, время, название)
- [ ] Увидеть кнопку "Добавить плейсхолдер"

### 1.2 Добавление плейсхолдера
- [ ] Нажать "Добавить плейсхолдер"
- [ ] Открылось модальное окно с заголовком "Добавить плейсхолдер"
- [ ] Видны поля:
  - [ ] Ключ плейсхолдера (input)
  - [ ] Отображаемое название (input)
  - [ ] Тип поля (select: text, textarea, date, number)
  - [ ] Текст подсказки (input)
  - [ ] Значение по умолчанию (input)
  - [ ] Обязательное поле (checkbox)

### 1.3 Валидация ключа
- [ ] Попробовать ввести ключ с пробелом: "мой ключ"
  - [ ] Должна появиться ошибка
- [ ] Попробовать ввести ключ с дефисом: "мой-ключ"
  - [ ] Должна появиться ошибка
- [ ] Попробовать ввести ключ с точкой: "мой.ключ"
  - [ ] Должна появиться ошибка
- [ ] Ввести правильный ключ: "мой_ключ"
  - [ ] Должно сохраниться без ошибок

### 1.4 Создание тестовых плейсхолдеров
Создать следующие плейсхолдеры:

**Плейсхолдер 1:**
- [ ] Ключ: `тест_текст`
- [ ] Название: "Тестовое текстовое поле"
- [ ] Тип: text
- [ ] Подсказка: "Введите любой текст"
- [ ] По умолчанию: (пусто)
- [ ] Обязательное: ✓
- [ ] Нажать "Сохранить"
- [ ] Плейсхолдер появился в списке

**Плейсхолдер 2:**
- [ ] Ключ: `тест_область`
- [ ] Название: "Тестовое многострочное поле"
- [ ] Тип: textarea
- [ ] Подсказка: "Можно ввести несколько строк"
- [ ] По умолчанию: (пусто)
- [ ] Обязательное: (не отмечено)
- [ ] Плейсхолдер появился в списке

**Плейсхолдер 3:**
- [ ] Ключ: `тест_дата`
- [ ] Название: "Тестовая дата"
- [ ] Тип: date
- [ ] Подсказка: "Выберите дату"
- [ ] По умолчанию: (пусто)
- [ ] Обязательное: ✓
- [ ] Плейсхолдер появился в списке

**Плейсхолдер 4:**
- [ ] Ключ: `тест_число`
- [ ] Название: "Тестовое число"
- [ ] Тип: number
- [ ] Подсказка: "Введите целое число"
- [ ] По умолчанию: "100"
- [ ] Обязательное: (не отмечено)
- [ ] Плейсхолдер появился в списке

### 1.5 Редактирование плейсхолдера
- [ ] Нажать "Редактировать" на любом плейсхолдере
- [ ] Открылось модальное окно с заголовком "Редактировать плейсхолдер"
- [ ] Поле "Ключ" заблокировано (readonly)
- [ ] Все остальные поля заполнены сохраненными значениями
- [ ] Изменить название на "Измененное название"
- [ ] Сохранить
- [ ] Название обновилось в списке

### 1.6 Удаление плейсхолдера
- [ ] Нажать "Удалить" на последнем плейсхолдере
- [ ] Плейсхолдер исчез из списка
- [ ] Остальные плейсхолдеры остались

### 1.7 Проверка дубликатов
- [ ] Создать плейсхолдер с ключом `дубликат`
- [ ] Попробовать создать еще один с ключом `дубликат`
- [ ] Должна появиться ошибка "Плейсхолдер с таким ключом уже существует"

## 2. Тестирование создания шаблона с плейсхолдерами

### 2.1 Подготовка тестового Word файла
- [ ] Создать файл test_template.docx с содержимым:
```
ТЕСТОВЫЙ ДОКУМЕНТ

Дата создания: {{дата}}
Время создания: {{время}}
Название: {{название}}

Текст: {{тест_текст}}
Многострочный текст: {{тест_область}}
Дата: {{тест_дата}}
Число: {{тест_число}}

Конец документа.
```

### 2.2 Создание шаблона
- [ ] Заполнить форму создания шаблона:
  - [ ] Название: "Тестовый шаблон с динамическими полями"
  - [ ] Категория: (любая)
  - [ ] Формат файла: Word (.docx)
  - [ ] Файл шаблона: загрузить test_template.docx
  - [ ] Плейсхолдеры уже добавлены (из шага 1.4)
- [ ] Сохранить шаблон
- [ ] Появилось сообщение об успешном создании
- [ ] Шаблон появился в списке шаблонов

---

## 3. Тестирование создания документа из шаблона

### 3.1 Открытие формы создания документа
- [ ] Перейти в "Документы" → "Создать из шаблона"
- [ ] Увидеть форму с полями:
  - [ ] Шаблон документа (select)
  - [ ] Название документа (input)

### 3.2 Выбор шаблона
- [ ] Выбрать "Тестовый шаблон с динамическими полями" из списка
- [ ] Страница перезагрузилась
- [ ] В URL появился параметр: `?template=X`
- [ ] Появился раздел "Заполнение полей шаблона"

### 3.3 Проверка динамических полей
Должны появиться следующие поля:

- [ ] **Тестовое текстовое поле** (обязательное)
  - [ ] Тип: text input
  - [ ] Подсказка: "Введите любой текст"
  - [ ] Помечено как обязательное (*)

- [ ] **Тестовое многострочное поле**
  - [ ] Тип: textarea (3 строки)
  - [ ] Подсказка: "Можно ввести несколько строк"
  - [ ] НЕ помечено как обязательное

- [ ] **Тестовая дата** (обязательное)
  - [ ] Тип: date picker
  - [ ] Подсказка: "Выберите дату"
  - [ ] Помечено как обязательное (*)

- [ ] **Тестовое число**
  - [ ] Тип: number input
  - [ ] Значение по умолчанию: 100
  - [ ] Подсказка: "Введите целое число"
  - [ ] НЕ помечено как обязательное

### 3.4 Заполнение формы
- [ ] Название документа: "Тестовый документ 001"
- [ ] Тестовое текстовое поле: "Привет, мир!"
- [ ] Тестовое многострочное поле: "Строка 1\nСтрока 2\nСтрока 3"
- [ ] Тестовая дата: "15.01.2024"
- [ ] Тестовое число: оставить 100 (по умолчанию)
- [ ] Ответственный: (выбрать пользователя)
- [ ] Срок исполнения: (выбрать дату)

### 3.5 Создание документа
- [ ] Нажать "Создать документ"
- [ ] Появилось сообщение об успешном создании
- [ ] Перенаправлен на страницу документа

---

## 4. Тестирование генерации файла

### 4.1 Проверка созданного документа
- [ ] На странице документа есть кнопка "Скачать сгенерированный файл"
- [ ] Нажать на кнопку
- [ ] Файл скачался с именем типа `test_2024_01_15_document.docx`

### 4.2 Проверка содержимого Word файла
Открыть скачанный файл в Microsoft Word или LibreOffice:

- [ ] Плейсхолдер `{{дата}}` заменен на текущую дату (формат ДД.ММ.ГГГГ)
- [ ] Плейсхолдер `{{время}}` заменен на время создания (формат ЧЧ:ММ)
- [ ] Плейсхолдер `{{название}}` заменен на "Тестовый документ 001"
- [ ] Плейсхолдер `{{тест_текст}}` заменен на "Привет, мир!"
- [ ] Плейсхолдер `{{тест_область}}` заменен на многострочный текст
- [ ] Плейсхолдер `{{тест_дата}}` заменен на "15.01.2024"
- [ ] Плейсхолдер `{{тест_число}}` заменен на "100"
- [ ] Никаких `{{плейсхолдеров}}` не осталось в документе

## 5. Тестирование валидации

### 5.1 Обязательные поля
- [ ] Создать новый документ из того же шаблона
- [ ] Оставить обязательное поле "Тестовое текстовое поле" пустым
- [ ] Попытаться создать документ
- [ ] Появилась ошибка валидации
- [ ] Документ НЕ создался

- [ ] Заполнить обязательное поле
- [ ] Оставить необязательное поле "Тестовое многострочное поле" пустым
- [ ] Создать документ
- [ ] Документ успешно создался
- [ ] В скачанном файле пустое поле заменено на "" (пустую строку)

### 5.2 Типы данных
- [ ] Попробовать ввести буквы в числовое поле
- [ ] Должно быть запрещено браузером или появится ошибка

- [ ] Выбрать дату в поле типа date
- [ ] Дата должна отформатироваться в ДД.ММ.ГГГГ в сгенерированном файле

## 6. Тестирование редактирования шаблона

### 6.1 Изменение плейсхолдеров существующего шаблона
- [ ] Открыть созданный шаблон на редактирование
- [ ] Плейсхолдеры отображаются в списке
- [ ] Добавить новый плейсхолдер:
  - [ ] Ключ: `новое_поле`
  - [ ] Название: "Новое поле"
  - [ ] Тип: text
  - [ ] Обязательное: нет
- [ ] Сохранить шаблон

- [ ] Создать новый документ из этого шаблона
- [ ] В форме появилось поле "Новое поле"
- [ ] Заполнить и создать документ
- [ ] Проверить, что новый плейсхолдер заменяется (если добавить `{{новое_поле}}` в Word файл)

## 7. Тестирование Edge Cases

### 7.1 Шаблон без плейсхолдеров
- [ ] Создать шаблон без добавления кастомных плейсхолдеров
- [ ] Сохранить
- [ ] Создать документ из этого шаблона
- [ ] Форма показывает только стандартные поля (название, ответственный, срок)
- [ ] Раздел "Заполнение полей шаблона" НЕ появляется
- [ ] Документ создается нормально

### 7.2 Переключение между шаблонами
- [ ] Открыть форму создания документа
- [ ] Выбрать шаблон А (с плейсхолдерами)
- [ ] Страница обновилась, показаны поля шаблона А
- [ ] Изменить выбор на шаблон Б (с другими плейсхолдерами)
- [ ] Страница обновилась, показаны поля шаблона Б
- [ ] Поля шаблона А исчезли

### 7.3 Несуществующий template_id
- [ ] Вручную открыть URL: `/documents/create_from_template/?template=99999`
- [ ] Форма загружается без ошибок
- [ ] Показаны только стандартные поля
- [ ] Можно выбрать шаблон из списка

## 8. Тестирование производительности

### 8.1 Много плейсхолдеров
- [ ] Создать шаблон с 15-20 плейсхолдерами
- [ ] Форма загружается без задержек
- [ ] Все поля отображаются корректно
- [ ] Документ создается без ошибок

### 8.2 Длинные значения
- [ ] Создать документ с очень длинным текстом в textarea (1000+ символов)
- [ ] Документ создается
- [ ] Весь текст корректно вставлен в Word файл

## 9. Тестирование безопасности

### 9.1 XSS защита
- [ ] Попробовать ввести в поле: `<script>alert('XSS')</script>`
- [ ] Создать документ
- [ ] В Word файле текст вставлен как обычный текст, НЕ выполнен как скрипт

### 9.2 SQL Injection защита
- [ ] Попробовать ввести в поле: `'; DROP TABLE documents; --`
- [ ] Создать документ
- [ ] Документ создается нормально
- [ ] Таблицы в базе данных не затронуты

---

## 10. Тестирование обратной совместимости

### 10.1 Старые шаблоны
Если есть старые шаблоны, созданные до внедрения динамических плейсхолдеров:

- [ ] Старые шаблоны отображаются в списке
- [ ] Можно создать документ из старого шаблона
- [ ] Форма показывает только стандартные поля
- [ ] Документ создается без ошибок
