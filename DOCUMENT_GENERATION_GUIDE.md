# Автоматическая генерация документов из шаблонов

## Описание системы

Система автоматической генерации документов позволяет создавать полноценные документы (договоры, акты, приказы и т.д.) с реалистичным содержимым в пару кликов.

## Установка и настройка

### 1. Установите необходимую библиотеку

```bash
pip install python-docx
```

### 2. Загрузите готовые шаблоны

Запустите management command для загрузки предустановленных шаблонов:

```bash
python manage.py load_templates
```

Эта команда создаст следующие шаблоны:
- **Трудовой договор** - полный договор с сотрудником со всеми разделами
- **Акт приема-передачи документов** - акт передачи документации между отделами
- **Приказ о приеме на работу** - приказ с детальными поручениями
- **Служебная записка** - внутренний документ для коммуникации
- **Протокол собрания** - протокол совещания с решениями
- **Договор поставки** - договор с поставщиком со всеми условиями

### 3. Как использовать

#### Способ 1: Через интерфейс шаблонов

1. Перейдите в раздел **"Шаблоны документов"**
2. Найдите нужный шаблон
3. Нажмите кнопку **"Создать документ"** (с иконкой волшебной палочки)
4. Заполните минимальные данные (имя, должность, отдел)
5. Нажмите **"Создать документ автоматически"**
6. Система сгенерирует DOCX файл с полным содержимым
7. Документ будет сохранен в системе и доступен для скачивания

#### Способ 2: Через главную страницу

1. На главной странице найдите раздел "Быстрые действия"
2. Выберите нужный тип документа
3. Следуйте инструкциям из Способа 1

## Что генерируется автоматически

### Для всех типов документов:
- Номера документов (формат: 001/2026)
- Текущая дата
- Реквизиты организации (ИНН, КПП, ОГРН, адрес, банковские реквизиты)
- Подписи и печати (места для них)
- Структура документа согласно стандартам

### Трудовой договор:
- Полный текст договора на ~3 страницы
- Все разделы (предмет, права и обязанности, оплата труда, режим работы)
- Реквизиты работодателя и работника
- Условия испытательного срока
- График работы
- Размер оклада

### Акт приема-передачи:
- Список передаваемых документов
- Количество документов и страниц
- Состояние документов
- Подписи передающей и принимающей стороны

### Приказ о приеме:
- Номер и дата приказа
- Данные о сотруднике
- Условия приема (должность, отдел, оклад)
- Поручения руководителям
- Основания (трудовой договор, заявление)

### Служебная записка:
- От кого / Кому
- Тема
- Содержание вопроса
- Просьбы и поручения
- Контактная информация

### Протокол собрания:
- Повестка дня
- Список присутствующих
- Выступления (СЛУШАЛИ / ВЫСТУПИЛИ / РЕШИЛИ)
- Принятые решения
- Поручения с указанием ответственных и сроков

### Договор поставки:
- Полный текст договора на ~4 страницы
- Предмет договора (список товаров)
- Условия поставки и оплаты
- Ответственность сторон
- Реквизиты обеих сторон

## Настройка под вашу организацию

### Изменение реквизитов

Откройте файл `documents/views.py`, функцию `generate_document_auto` и измените следующие данные:

```python
replacements = {
    '{director_name}': "ВАШ_ДИРЕКТОР",  # Измените на вашего директора
    '{work_address}': "ВАШ_АДРЕС",      # Ваш рабочий адрес
    # ... и т.д.
}
```

### Добавление собственных шаблонов

1. Откройте `documents/management/commands/load_templates.py`
2. Добавьте новый словарь в список `templates_data`:

```python
{
    'name': 'Ваш документ',
    'type': 'contract',  # contract, act, order, memo, protocol
    'file_format': 'docx',
    'description': 'Описание вашего документа',
    'content': '''
    Ваш шаблон с переменными {имя_переменной}
    '''
}
```

3. Запустите команду снова: `python manage.py load_templates`

## Преимущества системы

✅ **Экономия времени** - создание документа занимает 30 секунд вместо 30 минут
✅ **Реалистичное содержимое** - полноценные документы с правильной структурой
✅ **Автозаполнение** - все данные подставляются автоматически
✅ **Готовые шаблоны** - 6 типов документов из коробки
✅ **Хранение** - все документы сохраняются в системе
✅ **Скачивание** - DOCX файлы можно скачать и отредактировать
✅ **Гибкость** - легко добавить свои шаблоны

## Примеры использования

### Пример 1: Прием нового сотрудника

1. Создайте **"Трудовой договор"** - укажите ФИО, должность, оклад
2. Создайте **"Приказ о приеме на работу"** - укажите те же данные
3. Оба документа будут созданы с полным содержимым за минуту
4. Скачайте, распечатайте и подпишите

### Пример 2: Передача документов между отделами

1. Создайте **"Акт приема-передачи"**
2. Система автоматически заполнит список документов
3. Акт готов к печати и подписанию

### Пример 3: Закупка товаров

1. Создайте **"Договор поставки"**
2. Все условия, реквизиты и формулировки будут заполнены
3. Осталось только уточнить детали с поставщиком

## Поддержка
