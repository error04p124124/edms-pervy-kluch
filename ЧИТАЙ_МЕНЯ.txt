# ⚠️ ВАЖНАЯ ИНФОРМАЦИЯ О СИСТЕМЕ ШАБЛОНОВ

## 🎉 Обновление: Теперь поддерживаются реальные Office файлы!

Система шаблонов переделана для работы с **настоящими файлами Word, Excel и PowerPoint**.

### Что это значит?

**РАНЬШЕ:**
- Шаблоны были только в HTML формате
- Документы создавались как веб-страницы
- Сложно было распечатать или отредактировать

**ТЕПЕРЬ:**
- ✅ Создавайте шаблоны в Microsoft Word (.docx)
- ✅ Создавайте таблицы в Microsoft Excel (.xlsx)
- ✅ Создавайте презентации в PowerPoint (.pptx)
- ✅ Генерируйте готовые документы в один клик
- ✅ Скачивайте файлы и редактируйте в Office
- ✅ Все форматирование сохраняется!

### 📦 Требуется установка библиотек

**ВАЖНО:** Перед использованием системы выполните в командной строке:

```powershell
cd d:\12112
pip install python-docx openpyxl python-pptx Pillow
```

Затем примените миграции:
```powershell
python manage.py makemigrations
python manage.py migrate
```

Перезапустите сервер:
```powershell
python manage.py runserver 3000
```

### 📝 Как использовать?

**1. Создайте шаблон в Word:**
- Откройте Microsoft Word
- Напишите текст документа
- Вставьте плейсхолдеры: `{{имя}}`, `{{должность}}`, `{{дата}}`
- Сохраните как .docx

**2. Загрузите шаблон в систему:**
- Перейдите: **Шаблоны** → **Создать шаблон**
- Выберите формат "Word документ (.docx)"
- Загрузите ваш файл
- Сохраните

**3. Создайте документ:**
- Перейдите: **Документы** → **Создать из шаблона**
- Выберите шаблон
- Заполните поля (имя, должность и т.д.)
- Нажмите "Создать"

**4. Скачайте готовый документ:**
- Откройте карточку документа
- Найдите "Сгенерированный документ"
- Нажмите кнопку скачивания
- Откройте файл в Word - все заполнено!

### 🎯 Доступные плейсхолдеры

Используйте в шаблонах:
- `{{имя}}` - ФИО сотрудника
- `{{должность}}` - должность
- `{{отдел}}` - отдел/подразделение
- `{{дата}}` - текущая дата (автоматически)
- `{{время}}` - текущее время (автоматически)
- `{{custom1}}` - дополнительное поле 1
- `{{custom2}}` - дополнительное поле 2

### ❓ Частые вопросы

**Q: Старые HTML шаблоны перестанут работать?**
A: Нет, они продолжат работать. Новая функция добавлена, а не заменена.

**Q: Какие форматы поддерживаются?**
A: Word (.docx), Excel (.xlsx), PowerPoint (.pptx), PDF, HTML

**Q: Плейсхолдеры не заменяются в Word**
A: Убедитесь, что пишете `{{имя}}` слитно, одним куском текста (не копируйте из других мест)

**Q: Ошибка "Библиотека не установлена"**
A: Выполните команду установки библиотек (см. выше)

### 💡 Пример шаблона Word

```
                        ПРИКАЗ
                    от {{дата}}

    О приеме на работу

ПРИКАЗЫВАЮ:

1. Принять на работу {{имя}} на должность {{должность}}
   в {{отдел}} с {{дата}}.

2. Установить оклад согласно штатному расписанию.


Руководитель                    ______________
```

Сохраните это в Word как `приказ.docx` и загрузите в систему!
