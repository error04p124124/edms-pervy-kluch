# Система ЭДО "Первый ключ"

## Быстрый старт

### 1. Первый запуск (только один раз)

**Windows:**
```
install.bat
```

**Linux/Mac:**
```
chmod +x install.sh start_server.sh
./install.sh
```

### 2. Запуск сервера

**Windows:**
```
start_server.bat
```

**Linux/Mac:**
```
./start_server.sh
```

### 3. Вход в систему

Откройте браузер: http://127.0.0.1:8000

**Тестовые учетные записи:**
- Делопроизводитель: `clerk` / `clerk123`
- Руководитель: `manager` / `manager123`
- Сотрудник: `employee` / `employee123`

## Основные операции

### Для делопроизводителя

#### Создание документа из шаблона:
1. Главная → "Создать из шаблона"
2. Выберите шаблон
3. Заполните поля (ФИО, должность, отдел)
4. Укажите ответственного и срок
5. Нажмите "Создать документ"

#### Регистрация документа:
1. Откройте документ
2. Нажмите "Зарегистрировать"
3. Документ получит уникальный номер ГГГГ/ММ/НННН

#### Настройка маршрута согласования:
1. Откройте документ
2. Нажмите "Маршрут согласования"
3. Выберите согласующих в нужном порядке
4. Сохраните

#### Создание шаблона:
1. Меню → Шаблоны
2. "Создать шаблон"
3. Укажите тип и название
4. Введите HTML с плейсхолдерами:
   - {{имя}} - ФИО
   - {{должность}} - Должность
   - {{отдел}} - Отдел
   - {{дата}} - Текущая дата
   - {{custom1}}, {{custom2}} - Дополнительные поля

#### Просмотр отчетов:
1. Меню → Отчеты
2. Просмотр статистики
3. Печать: Ctrl+P

### Для руководителя

#### Согласование документа:
1. Открыть документ
2. Выбрать "Утвердить" или "Отклонить"
3. Добавить комментарий (опционально)
4. Нажать "Отправить"

### Для сотрудника

#### Создание заявки:
1. Главная → "Создать из шаблона"
2. Выбрать шаблон (например, "Заявление на отпуск")
3. Заполнить данные
4. Отправить

## Фильтры документов

В списке документов доступны фильтры:
- **Поиск** - по названию или номеру
- **Статус** - черновик, на согласовании, утвержден и т.д.
- **Тип** - приказ, договор, акт и т.д.
- **Дата** - от/до
- **Только просроченные** - показать просроченные

## Массовые операции (только для делопроизводителя)

1. В списке документов отметьте нужные чекбоксами
2. Нажмите "Архивировать выбранные"

## Уведомления

- Колокольчик в правом верхнем углу
- Красный индикатор - новые уведомления
- Клик по уведомлению - переход к документу

## Редактирование профиля

1. Меню → Имя пользователя → Профиль
2. Измените данные
3. Загрузите аватар (опционально)
4. Сохраните

## Статусы документов

- **Черновик** - документ создан, но не зарегистрирован
- **На согласовании** - проходит согласование
- **Утвержден** - согласован и утвержден
- **Отклонен** - отклонен на этапе согласования
- **В архиве** - перемещен в архив

## Советы

1. **Автосохранение номеров:** Документы получают номера автоматически при регистрации
2. **Просроченные документы:** Подсвечиваются красным в списках
3. **История изменений:** Все действия логируются и доступны в карточке документа
4. **Печать отчетов:** Используйте Ctrl+P на странице отчетов

## Решение проблем

### Забыли пароль?
Обратитесь к администратору для сброса пароля

### Не видите нужные кнопки?
Проверьте свою роль в профиле. Некоторые функции доступны только делопроизводителям.

### Документ не сохраняется?
- Проверьте обязательные поля (отмечены *)
- Убедитесь, что заполнили все требуемые данные

### Не приходят уведомления?
Уведомления появляются в системе (колокольчик вверху). Email-уведомления требуют дополнительной настройки.

## Горячие клавиши

- **Ctrl+P** - Печать текущей страницы
- **Escape** - Закрыть модальные окна

## Контакты поддержки

При технических проблемах обращайтесь к системному администратору.
