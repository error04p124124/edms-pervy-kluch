## 🎯 Возможности системы

### Что можно делать:

1. **Создавать шаблоны в Office:**
   - Создайте документ в Word/Excel/PowerPoint
   - Добавьте плейсхолдеры: `{{имя}}`, `{{должность}}` и т.д.
   - Загрузите в систему

2. **Генерировать документы:**
   - Выберите шаблон
   - Заполните данные в форме
   - Система автоматически создаст готовый файл

3. **Скачивать готовые файлы:**
   - Файл генерируется в оригинальном формате
   - Сохраняется все форматирование
   - Можно редактировать в Office

### Поддерживаемые плейсхолдеры:

| Плейсхолдер | Откуда берется | Пример |
|-------------|----------------|---------|
| `{{имя}}` | Форма создания | Иванов И.И. |
| `{{название}}` | Поле "Название документа" | Приказ о приеме |
| `{{должность}}` | Форма создания | Менеджер |
| `{{отдел}}` | Форма создания | Отдел продаж |
| `{{дата}}` | Автоматически | 20.01.2026 |
| `{{время}}` | Автоматически | 14:30 |
| `{{custom1}}` | Доп. поле 1 | Любой текст |
| `{{custom2}}` | Доп. поле 2 | Любой текст |

## 🔄 Процесс работы

### Создание документа из шаблона:

```
1. Пользователь выбирает шаблон
         ↓
2. Заполняет форму с данными
         ↓
3. Система создает Document в БД
         ↓
4. office_utils.generate_document_from_template()
   - Открывает файл-шаблон
   - Заменяет плейсхолдеры
   - Сохраняет в media/generated/
         ↓
5. Путь сохраняется в document.generated_file
         ↓
6. Пользователь может скачать файл
```

### Скачивание файла:

```
1. Пользователь нажимает "Скачать"
         ↓
2. Проверка прав доступа
         ↓
3. Проверка наличия файла
         ↓
4. Определение MIME-типа
         ↓
5. FileResponse с правильными заголовками
         ↓
6. Логирование в DocumentHistory
         ↓
7. Файл загружается в браузер
```

## 🐛 Известные ограничения

1. **Word документы:**
   - Плейсхолдер должен быть написан слитно (не копировать из других мест)
   - Не поддерживается замена в SmartArt и диаграммах

2. **Excel таблицы:**
   - Замена только в текстовых ячейках
   - Не заменяется в формулах

3. **PDF:**
   - Загрузка возможна, но генерация пока не реализована
   - PDF создается статическим (без замены плейсхолдеров)
