## 🏁 Готовность к запуску

### Что нужно сделать:

1. **Установка зависимостей:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Применение миграций:**
   ```bash
   python manage.py makemigrations
   python manage.py migrate
   ```

3. **Создание суперпользователя:**
   ```bash
   python manage.py createsuperuser
   ```

4. **Инициализация тестовых данных:**
   ```bash
   python init_data.py
   ```

5. **Запуск сервера:**
   ```bash
   python manage.py runserver
   ```

### Или использовать автоматические скрипты:

**Windows:**
```bash
install.bat      # Делает пункты 1-4
start_server.bat # Запускает сервер
```

**Linux/Mac:**
```bash
chmod +x install.sh start_server.sh
./install.sh      # Делает пункты 1-4
./start_server.sh # Запускает сервер
```
