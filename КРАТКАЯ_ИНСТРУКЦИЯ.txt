# КРАТКО: Динамические плейсхолдеры

## Что сделано

Теперь при создании шаблона можно самостоятельно добавлять любые плейсхолдеры (поля для заполнения).

## Как использовать

### 1. При создании/редактировании шаблона:

- Найдите раздел "Управление плейсхолдерами"
- Нажмите "Добавить плейсхолдер"
- Заполните:
  - **Ключ**: имя переменной (например: `фио`, `адрес`, `телефон`)
  - **Название**: что увидит пользователь ("ФИО сотрудника", "Адрес проживания")
  - **Тип**: text (строка), textarea (текст), date (дата), number (число)
  - **Подсказка**: текст под полем (опционально)
  - **По умолчанию**: предзаполненное значение (опционально)
  - **Обязательное**: галочка, если поле обязательно

### 2. В файле шаблона (Word/Excel/PowerPoint):

Используйте `{{ключ}}` там, где нужна замена:

```
Договор от {{дата}}

Работник: {{фио}}
Адрес: {{адрес}}
Телефон: {{телефон}}
```

### 3. При создании документа:

- Выберите шаблон
- Страница обновится и покажет ваши поля
- Заполните их
- Создайте документ

Готово! Файл создастся с заменой всех `{{плейсхолдеров}}` на введенные значения.

## Автоматические плейсхолдеры

Всегда доступны без настройки:

- `{{дата}}` - текущая дата (ДД.ММ.ГГГГ)
- `{{время}}` - текущее время (ЧЧ:ММ)
- `{{название}}` - название документа

## Пример

**Шаблон "Трудовой договор":**

Плейсхолдеры:
- `фио_работника` (text, обязательное) → "ФИО работника"
- `должность` (text, обязательное) → "Должность"
- `оклад` (number, обязательное) → "Оклад (руб.)"
- `дата_начала` (date, обязательное) → "Дата начала работы"

В Word файле:
```
ТРУДОВОЙ ДОГОВОР от {{дата}}

Работник: {{фио_работника}}
Должность: {{должность}}
Оклад: {{оклад}} руб.
Дата начала: {{дата_начала}}
```

При создании документа появятся поля:
- ФИО работника: [текстовое поле]
- Должность: [текстовое поле]
- Оклад (руб.): [числовое поле]
- Дата начала работы: [выбор даты]

## Важно

✅ **Можно:**
- Использовать русские и английские буквы в ключах
- Использовать цифры и подчеркивания
- Создавать сколько угодно плейсхолдеров

❌ **Нельзя:**
- Использовать пробелы, дефисы, точки в ключах
- Дублировать ключи `дата`, `время`, `название` (они автоматические)
- Использовать спецсимволы в ключах

## Миграция не требуется

- База данных не меняется
- Старые шаблоны продолжают работать
- Новые шаблоны используют динамическую систему

## Если что-то не работает

1. **Плейсхолдер не заменяется?**
   - Проверьте точное совпадение ключа
   - Используйте `{{ключ}}` в шаблоне

2. **Не вижу свои поля?**
   - Обновите страницу (F5) после выбора шаблона
   - Убедитесь, что плейсхолдеры сохранены в шаблоне

3. **Поле не обязательное, хотя должно быть?**
   - Проверьте галочку "Обязательное поле"
   - Пересохраните шаблон
