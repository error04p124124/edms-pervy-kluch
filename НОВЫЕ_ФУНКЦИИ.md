### 📋 1. Расширенный Workflow документов

**Новые статусы документов:**
- ✅ **Черновик** (draft)
- ✅ **Отправлен на согласование** (sent_for_approval)

## 🚀 Как использовать

### 1. Workflow документов

```python
# В коде или через админку
document.status = 'sent_for_approval'
document.approval_type = 'sequential'  # или 'parallel'
document.save()

# Продвинуть документ по workflow
from documents.utils import advance_workflow
advance_workflow(document, user, decision='approved', comment='Согласовано')
```

### 2. Создание версии

```python
from documents.utils import create_document_version
version = create_document_version(document, user, 'Исправлены опечатки')
```

### 3. Подписание ЭП

```python
from documents.utils import sign_document
signature = sign_document(document, user, request)
```

### 4. Отправка уведомления

```python
from documents.utils import send_notification
send_notification(user, 'Документ готов', document)
```

### 5. Создание задачи

```python
from documents.utils import create_task_from_document
task = create_task_from_document(document, 'approve', assigned_to, created_by, deadline)
```

### 6. Логирование

```python
from documents.utils import log_audit
log_audit(user, 'create', 'Document', document.id, str(document), 'Создан новый документ', request)
```
