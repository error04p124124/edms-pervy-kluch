📌 ВАЖНО: Перед использованием системы шаблонов выполните установку библиотек!

## Установка Office библиотек

```powershell
pip install python-docx openpyxl python-pptx Pillow
```

Или используйте файл requirements:
```powershell
pip install -r office_requirements.txt
```

## Применение миграций

После установки библиотек обязательно выполните:
```powershell
python manage.py makemigrations
python manage.py migrate
```

Это добавит новые поля в модели DocumentTemplate и Document для работы с Office файлами.

## Что изменилось

✅ **Шаблоны теперь поддерживают реальные файлы Office**
- Word (.docx) - документы, приказы, письма
- Excel (.xlsx) - таблицы, реестры, отчёты
- PowerPoint (.pptx) - презентации
- PDF (.pdf) - документы для печати
- HTML (.html) - веб-документы

✅ **Автоматическая генерация документов**
При создании документа из шаблона система:
1. Берёт файл-шаблон
2. Заменяет все плейсхолдеры ({{имя}}, {{дата}} и т.д.)
3. Сохраняет готовый файл
4. Позволяет скачать его

✅ **Все форматирование сохраняется**
Стили, шрифты, таблицы, изображения из шаблона остаются как есть
