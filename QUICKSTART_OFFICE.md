## Шаг 4: Создание первого шаблона

### Вариант А: Word шаблон

1. Откройте Word, создайте документ:
   ```
   ПРИКАЗ
   от {{дата}}
   
   Приказываю принять на работу {{имя}}
   на должность {{должность}} в {{отдел}}.
   
   Руководитель _____________
   ```

2. Сохраните как `приказ.docx`

3. В системе:
   - Перейдите **Шаблоны** → **Создать шаблон**
   - Название: "Приказ о приеме"
   - Тип: "Приказ"
   - Формат: "Word документ (.docx)"
   - Загрузите файл `приказ.docx`
   - Сохраните

### Вариант Б: Excel шаблон

1. Откройте Excel, создайте таблицу:
   | Поле | Значение |
   |------|----------|
   | ФИО | {{имя}} |
   | Должность | {{должность}} |
   | Отдел | {{отдел}} |
   | Дата | {{дата}} |

2. Сохраните как `реестр.xlsx`

3. В системе:
   - Перейдите **Шаблоны** → **Создать шаблон**
   - Название: "Реестр сотрудников"
   - Тип: "Другое"
   - Формат: "Excel таблица (.xlsx)"
   - Загрузите файл `реестр.xlsx`
   - Сохраните

## Шаг 5: Создание документа из шаблона

1. Перейдите **Документы** → **Создать из шаблона**
2. Выберите созданный шаблон
3. Заполните поля:
   - Название: "Приказ о приеме Иванова"
   - ФИО: "Иванов Иван Иванович"
   - Должность: "Менеджер"
   - Отдел: "Отдел продаж"
4. Нажмите **Создать**

Система автоматически создаст файл с заменёнными данными!

## Шаг 6: Скачивание документа

1. Откройте созданный документ
2. Найдите раздел **"Сгенерированный документ"**
3. Нажмите **Скачать Word документ** (или Excel таблица)
4. Откройте скачанный файл - все плейсхолдеры заменены!
