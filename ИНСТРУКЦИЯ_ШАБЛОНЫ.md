# Инструкция по работе с шаблонами документов

## Обновление системы

В системе отключена функция автоматического создания документов из шаблонов из-за технических проблем.

## Как работать с шаблонами сейчас:

### 1. Загрузка шаблонов
- Перейдите в раздел **"Шаблоны документов"**
- Нажмите **"Создать шаблон"**
- Заполните форму:
  - Название шаблона
  - Тип документа (договор, акт, счет и т.д.)
  - Формат файла (.docx, .xlsx, .pptx, .pdf)
  - Загрузите файл шаблона
  - Описание (необязательно)
- Нажмите **"Сохранить"**

### 2. Скачивание шаблонов
- Перейдите в раздел **"Шаблоны документов"**
- Найдите нужный шаблон
- Нажмите кнопку **"Скачать"** (зеленая кнопка с иконкой загрузки)
- Файл шаблона будет загружен на ваш компьютер

### 3. Создание документа на основе шаблона

**Создание документа выполняется вручную:**

1. Скачайте нужный шаблон (см. п.2)
2. Откройте скачанный файл в соответствующей программе (Word, Excel и т.д.)
3. **Заполните документ вручную:**
   - Измените все места, где нужно указать конкретные данные
   - Укажите имена, даты, суммы, номера и другую информацию
   - Проверьте правильность заполнения
   - Убедитесь, что все необходимые поля заполнены
4. Сохраните заполненный документ под новым именем
5. Загрузите готовый документ в систему:
   - Перейдите в **"Документы"**
   - Нажмите **"Создать документ"**
   - Заполните название
   - Прикрепите готовый файл
   - Назначьте ответственного (при необходимости)
   - Установите срок (при необходимости)
   - Нажмите **"Сохранить"**

#### Что изменять в шаблоне:

При создании шаблонов рекомендуется использовать понятные маркеры для мест, которые нужно заполнить:
- `[ИМЯ]` - для имени сотрудника/контрагента
- `[ДАТА]` - для даты документа
- `[СУММА]` - для денежных сумм
- `[НОМЕР]` - для номеров документов
- `[ДОЛЖНОСТЬ]` - для должности
- И т.д.

При заполнении документа просто замените эти маркеры на реальные данные.

### 4. Редактирование шаблонов
- Перейдите в раздел **"Шаблоны документов"**
- Найдите нужный шаблон
- Нажмите кнопку **"Изменить"** (желтая кнопка)
- Внесите изменения
- Нажмите **"Сохранить"**

### 5. Удаление шаблонов
- Перейдите в раздел **"Шаблоны документов"**
- Найдите ненужный шаблон
- Нажмите кнопку **"Удалить"** (красная кнопка)
- Подтвердите удаление

## Преимущества текущего подхода:

✅ **Простота** - понятный процесс без технических сложностей
✅ **Гибкость** - можно вносить любые изменения в документ
✅ **Надежность** - нет ошибок при создании документов
✅ **Контроль** - полный контроль над содержимым документа

## Важно:

- Шаблоны хранятся в системе и доступны всем пользователям
- Файлы шаблонов можно обновлять при необходимости
- Готовые документы загружаются в систему как обычные файлы
- Все документы проходят через стандартный процесс согласования
