# ШПАРГАЛКА ДЛЯ АДМИНИСТРАТОРОВ

## Создание шаблона с динамическими плейсхолдерами

### Шаг 1: Откройте форму создания шаблона
```
Навигация: Шаблоны документов → Создать шаблон
```

### Шаг 2: Добавьте плейсхолдеры

| Что заполнить | Что это значит | Пример |
|---------------|----------------|--------|
| **Ключ** | Имя переменной в шаблоне | `фио_сотрудника` |
| **Название** | Подпись поля для пользователя | "ФИО сотрудника" |
| **Тип** | Вид поля ввода | text / textarea / date / number |
| **Подсказка** | Текст под полем (опционально) | "Введите полное ФИО" |
| **По умолчанию** | Предзаполненное значение (опционально) | "Не указано" |
| **Обязательное** | Галочка если нужна валидация | ✓ |

### Шаг 3: Создайте файл шаблона

**Для Word (.docx):**
```
Договор №_____ от {{дата}}

Работник: {{фио_сотрудника}}
Должность: {{должность}}
Адрес: {{адрес_проживания}}
```

**Для Excel (.xlsx):**
```
| Поле      | Значение                |
|-----------|-------------------------|
| ФИО       | {{фио_сотрудника}}      |
| Должность | {{должность}}           |
| Дата      | {{дата_приема}}         |
```

### Шаг 4: Загрузите и сохраните
- Выберите формат файла (Word/Excel/PowerPoint)
- Загрузите файл шаблона
- Нажмите "Сохранить"

---

## Типы полей

| Тип | Когда использовать | Пример данных |
|-----|-------------------|---------------|
| **text** | Короткий текст (1 строка) | ФИО, должность, город |
| **textarea** | Длинный текст (несколько строк) | Адрес, описание, примечание |
| **date** | Дата | Дата приема, дата рождения |
| **number** | Целое число | Оклад, количество, номер |

---

## Автоматические плейсхолдеры (всегда доступны)

```
{{дата}}     → 15.01.2024       (текущая дата)
{{время}}    → 14:30            (текущее время)
{{название}} → "Трудовой договор" (название документа)
```

## ✅ DO (Делайте так)

✅ Используйте понятные ключи:
   - `фио_директора`
   - `номер_счета`
   - `дата_приема`

✅ Давайте подробные названия:
   - "ФИО руководителя организации"
   - "Номер расчетного счета"
   - "Дата приема на работу"

✅ Добавляйте подсказки для сложных полей:
   - Паспорт: "Формат: 1234 567890"
   - Телефон: "Формат: +7 (999) 999-99-99"
   - ИНН: "10 или 12 цифр"

✅ Используйте правильный тип поля:
   - Адрес → `textarea`
   - Дата → `date`
   - Сумма → `number`
   - ФИО → `text`

✅ Отмечайте обязательные поля:
   - Критичные данные → обязательные
   - Дополнительная информация → необязательные

---

## ❌ DON'T (Не делайте так)

❌ Не используйте спецсимволы в ключах:
   - ❌ `моё-поле`
   - ❌ `моё.поле`
   - ❌ `моё поле`
   - ✅ `моё_поле` или `moe_pole`

❌ Не дублируйте автоматические плейсхолдеры:
   - ❌ Не создавайте свой `дата`
   - ❌ Не создавайте свой `время`
   - ❌ Не создавайте свой `название`

❌ Не создавайте слишком много полей:
   - ❌ 20+ полей в одном шаблоне
   - ✅ 5-10 полей оптимально
   - ✅ Разделите на несколько шаблонов если нужно больше

❌ Не используйте неинформативные имена:
   - ❌ `field1`, `x`, `temp`, `data`
   - ✅ `фио_сотрудника`, `адрес_доставки`, `номер_договора`

## 🔧 Устранение проблем

### Проблема: Плейсхолдер не заменяется

**Решение:**
1. Проверьте точное совпадение ключа
2. Используйте `{{ключ}}` в файле
3. Убедитесь, что нет опечаток

**Пример:**
```
❌ В шаблоне: {{фио}}
   В форме: фио_сотрудника
   → Не совпадает!

✅ В шаблоне: {{фио_сотрудника}}
   В форме: фио_сотрудника
   → Совпадает!
```

---

### Проблема: Не вижу свои поля

**Решение:**
1. После выбора шаблона страница должна обновиться
2. Обновите вручную (F5)
3. Проверьте, что плейсхолдеры сохранены

---

### Проблема: Поле не обязательное

**Решение:**
1. Откройте шаблон на редактирование
2. Найдите плейсхолдер
3. Нажмите "Редактировать"
4. Поставьте галочку "Обязательное поле"
5. Сохраните

## 📝 Быстрые команды

### Создать текстовое поле:
```
Ключ: название_поля
Название: Красивое название
Тип: text

```

```

Тип: textarea

Обязательное: ✓
```

```
Ключ: дата_документа
Название: Дата документа
Тип: date
Обязательное: ✓
```

### Создать поле для суммы:
```
Ключ: сумма
Название: Сумма (руб.)
Тип: number
По умолчанию: 0
Обязательное: ✓
```
